About

FramerFlow is an editor which helps you to create prototypes without any limits on windows platform.

Why does FramerFlow exist?

I love Framer.js but as an windows developer I could not find any good tool for Framer.js on Windows. There are someone like me on windows platform. So, for my personal use, , and also for someone, I made it.

When did FramerFlow start?

I started the code which became FramerFlow in the spring of 2015.
